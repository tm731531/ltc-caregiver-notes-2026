## 🌅 上午場 · 家庭照顧者的壓力與心理支持

> 主線：照顧者**為什麼會累垮 → 如何被接住**。橫跨核心課程單元 6、8、22。

### 單元 22｜家務處理協助技巧
家務處理的工作內容（9 項）：
1. 環境與用物清潔　2. 膳食協助　3. 文書服務　4. 代辦事宜　5. 身體清潔
6. 代購服務　7. 支持與關懷　8. 陪同就醫　9. 休閒性服務

> **延伸 · 這 9 項的重點**：這是照服員「家務」的**服務範圍**。官方學習目標另有**基本原則**：以案主為中心、「協助」而非「替代」（維持案主自立能力）、尊重案主原有生活習慣。家務不是幫傭代勞，是**支持案主維持生活功能**——跟下午「復能、支持自立」精神一致。

### 單元 6｜認識家庭照顧者與服務技巧
**家庭主要照顧者的型態**
- 家庭主要照顧者仍有 **80% 來自家庭**
- **70~80% 為家中女性**，其中以配偶為主
- 照顧者年齡 **60 歲以上者達 23~45%**
- 照顧者本身有慢性病者達 **33~46%**
- 多為女兒或媳婦（常有多重角色）

→ 核心現象：**「老人照顧老人，病人照顧病人」**

### 單元 8｜心理健康與壓力調適
**壓力是什麼**
- **壓力** = 人經歷生活環境變化（生活波折、重大改變事件）時，伴隨產生的感受反應，可能是緊張或焦慮。
- 壓力**並非完全不好**；但若生理與心理的負荷**超過個人能承受的量**，又缺少宣洩與支援管道，就會產生**不安、焦慮、害怕和憤怒**。

**主要照顧者的壓力來源（6 面向）**
| 面向 | 內容 |
|------|------|
| 工作 | 原先的工作 ＋ 照顧病人的工作（雙重負擔）|
| 角色 | 原來角色 ＋「全職照顧角色」→ 難兼顧 |
| 職業發展 | 喪失成就感（為照顧而辭職或放棄升遷）|
| 人際關係 | 缺少社會支持、被不諒解；缺少私人時間 |
| 社交 | 少參與團體活動 → 缺乏團體性共識 |
| 個人因素 | 作息受限、照顧知識不足、覺得照顧沒回報 |

→ 結果：**挫折忍受力、問題解決能力降低**

**主要照顧者的心理社會反應**：負荷大小因**患者疾病程度、照顧時間長短、患者合作程度、預後狀況**而異。照顧慢性病人或失智老人者，主要承受六種負荷：
1. 心理社會負荷　2. 生理負荷　3. 人際關係負荷　4. 病患依賴負荷　5. 社會羞恥負荷　6. 負性情緒負荷

> **延伸 ·「社會羞恥負荷」特別重要**：許多失智症家屬會因患者行為（在外遊走、失禁、罵人、重複問話）感到丟臉，於是**不敢帶出門、不敢讓外人知道**，把自己關起來硬撐。這正是下面「五大迷思」第 5 條要破解的。

**家庭照顧者的五大迷思**（來源：中華民國家庭照顧者關懷總會）
- ❌ 1. 照顧就應該自我犧牲？→ ✅ 唯有適當的**喘息**，才有更好的照護品質。
- ❌ 2. 100 分的照顧才是好照顧？→ ✅ 唯有**我快樂，才會有健康家人**。
- ❌ 3. 我最了解他，我來照顧才最好？→ ✅ 要**大方接受協助**，支援愈多、事半功倍。
- ❌ 4. 照顧工作是沒有價值的？→ ✅ 照顧家人有**重大意義與貢獻**，值得肯定。
- ❌ 5. 照顧是家事，不要讓外人知道？→ ✅ 應和**相同經驗的人交流**。

> 💡 第 5 條正好破解前面的「社會羞恥負荷」：別憋著，去找同溫層。

**照顧者需要的支持（三支柱）**
| 類型 | 內容 |
|------|------|
| （一）社會情感支持 | 被傾聽、被肯定尊重、隨時可求助的安全感、防其崩潰、協助面對死亡與哀傷 |
| （二）工具性支持 | 協助家務三餐、家人輪替、有人陪伴、經濟援助、交通協助 |
| （三）資訊性支持 | 提供資訊與社會資源、教照顧方法、協助處理病人嚴重問題行為 |

**協助照顧者因應壓力的方法（四招）**
1. **協助紓解壓力**：做好本份、找紓壓活動、找人分擔
2. **協助問題解決**：辨認問題 → 找解方
3. **提供情緒支持**：肯定稱讚、確認經驗、對未來的希望
4. **提供所需資訊**：瞭解需求、資源介紹 ＋ **向督導回報**

**長照 2.0 · 家庭照顧者服務據點**：① 心理諮商　② 舒壓課程　③ 做家庭照顧者的**後盾**

## 🕑 下午場 · 使命願景與照顧倫理法律

> 講師以組織經營與法律框架，帶「照顧為什麼要有方向感、法律怎麼保護被照顧者」。主要對應核心課程單元 1（相關法律基本認識）。

### 一、使命與願景 — 著力點和方向感
講師手繪 **McKinsey 7S 架構**：七要素環環相扣，以**共享價值（Shared Value）為中心**。

| 要素 | 中文 |
|------|------|
| **Shared Value（中心）** | 分享價值 / 共享價值 |
| Skill | 技能 |
| Style | 風格 |
| Staff | 人才 |
| Structure | 結構 |
| System | 系統 |
| Strategy | 策略 |

- **（一）使命**：養生送死無憾（讓生者好好活、逝者無憾）
- **（二）願景**：生命價值、人性尊嚴

### 二、照顧服務與「法益」
**法益** = 法律所保護的利益。人格法益由重到輕的階梯：
**生命 › 身體 › 健康 › 自由 › 名譽 › 財產權**

侵害對應：回復原狀 / 損害賠償。課堂討論題：Q1 甲的遺產如何分配（繼承）；Q2 某人可否結婚（近親結婚限制）。

### 三、認事用法正確委託 — 個案分析（家系圖）
**案情（依錄音重建）**
- 甲、乙為夫妻，育有**獨子丙**
- 丙與**丁（媳婦）**結婚
- 丙**超速駕車車禍身亡**；死亡時**丁已懷有胎兒**
- 乙（丙之母）亦已過世
- 喪偶的公公**甲**與守寡的媳婦**丁**日久生情

| 問題 | 法律解析 |
|------|----------|
| **Q1** 遺產如何繼承分配？ | **胎兒有繼承權**。民法第 7 條：胎兒以將來非死產者為限，關於利益之保護「**視為既已出生**」；故丙的遺產，胎兒與配偶丁等同為繼承人（依應繼分分配，注意特留分）。|
| **Q2** 甲（公公）與丁（媳婦）能否結婚？ | **不能**。公公與媳婦為**直系姻親**，民法第 983 條禁止結婚；且該限制「**於姻親關係消滅後亦適用之**」——即使丙已死、姻親關係消滅，甲丁仍不得結婚。|

> ⚠️ **重建說明**：此案情由錄音逐字稿重建；講師語速快、穿插故事，部分人名/細節音辨有誤差（丁的姓名尤其不清）。**法律結論（胎兒繼承、直系姻親禁婚）是標準民法、可信**；確切應繼分比例與姓名以講義為準。

> 🎙️ **錄音救回 · 照服員職業倫理警告**：講師延伸一個**現場真實警告**——居家照服員常遇到案主（如退休校長、獨居長者）求婚。若照服員與案主結婚以繼承遺產，社會譏為「**黑寡婦**」，法律上可能被視為**脫法行為**。照服員應守**專業分際**，不要「白天是照顧服務員、晚上變伴侶」。（投影片沒有、只在口述中，考倫理實務很受用。）

### 四、敏感性資料與隱私權保障
- **依據**：111 年憲判字 13 號判決
- **爭點**：個人資料保護

| 面向 | 內容 |
|------|------|
| 事前控制權 | 資料被蒐集前的同意與控制 |
| 事後控制權 | 資料被蒐集後的查詢、更正、刪除 |
| 當事人同意 | 須徵得當事人同意 |
| 公益與私權平衡 | 手段：① 去聯結（去識別化）② 告知同意 |
| 主管機關稽核 | 由主管機關監督 |

**兩則大法官解釋**：釋字 **603 號**（資訊自主權）、釋字 **293 號**（銀行個人存款隱私）。
**結語**：資訊是無形智財；唯有「**Fair Use**」合理使用才能逾越隱私保護的界線。

> 💡 **實務連結**：任何處理長輩**敏感健康資料**的系統（影像辨識、健康紀錄自動化等），白板的「**去聯結（去識別化）＋ 告知同意**」正是設計要守的紅線，不只是考試知識。

## 🧍 單元 10 · 身體結構與功能（錄音救回）

> 此單元投影片沒拍到，內容由講師吳孟明的錄音逐字稿整理。核心：人體由小到大的組成與各大系統。

**人體的組成層次**：細胞 › 組織 › 器官 › 系統 › 個體
（細胞是最小單位 → 相同細胞組成組織 → 組織構成器官 → 器官協同成系統 → 系統構成完整個體。）

**人體主要系統**
| 系統 | 主要功能（照顧相關）|
|------|--------------------|
| 骨骼與肌肉系統 | 支撐、活動、保護；關係到移位、翻身、跌倒、關節活動 |
| 循環系統（心臟、血管）| 輸送血液、氧氣；關係到血壓、脈搏 |
| 呼吸系統 | 氣體交換；關係到呼吸、抽痰、氧氣使用 |
| 消化系統 | 營養吸收、排便；關係到餵食、灌食、便秘 |
| 泌尿系統 | 排尿、代謝廢物；關係到尿管照護、失禁 |
| 神經系統 | 感覺與動作控制；關係到中風、失智、意識評估 |
| 內分泌系統 | 荷爾蒙調節；關係到糖尿病等慢性病 |
| 皮膚（外皮系統）| 保護、體溫調節；關係到壓瘡預防、清潔 |

> **延伸 · 為什麼照服員要懂身體結構**：懂系統才知道「為什麼要這樣照顧」——知道皮膚會壓瘡 → 才會定時翻身；懂泌尿系統 → 才注意尿量與尿管；懂神經系統 → 才理解中風/失智者的行為。身體結構是後面所有照顧技術的**底層知識**。

## 📚 延伸補充（我查證後補的，非課堂原文）

**延伸 1 · 家庭照顧者可用的求助管道**
- **家庭照顧者關懷專線 0800-50-7272**（諧音「有你，我好幸福」）— 衛福部設置、家總營運。社工諮詢、支持團體、心理協談。週一~五 9:00–17:00。
- **長照專線 1966** — 各縣市長照管理中心評估，可申請**喘息服務**、居家服務、日照等。
- **各縣市「家庭照顧者支持服務據點」** — 即長照 2.0 據點，提供心理諮商與舒壓課程。

**延伸 2 · 近親結婚限制（民法第 983 條）**：禁止結婚的親屬包括——
- **直系血親**及**直系姻親**（父母子女、祖孫）
- **旁系血親六親等以內**（兄弟姊妹、堂/表兄弟姊妹）
- **旁系姻親五親等以內**且輩分不相同者
- 立法理由：優生學與倫理道德。確切親等以民法 983 原文為準。

**延伸 3 · 兩則釋字**
- **釋字 603 號**：確立**資訊隱私權（資訊自主權）**受憲法保障（源於按捺指紋換身分證爭議）。
- **釋字 293 號**：涉及**銀行對客戶存款資料的保密義務**與公開的界線。

**延伸 4 · 7S 對應長照工作**：共享價值＝「照顧無價的生命」信念；策略/結構/系統＝服務流程與制度；人才/技能/風格＝照服員能力與服務態度。七者對齊，照顧才有一致方向。

**延伸 5 · 包紮的目的（今天那題釐清）**：急救包紮標準目的＝**固定敷料、壓迫止血、保暖、支持固定受傷部位**。「矯正畸形」**不是**包紮目的（那是石膏、副木、矯正器的功能）。
