## 114/06/13 兩處關鍵修正（摘自對照表，考生必知）

1. **應檢人須知 新增第十四點（受傷處理）**：測試中輕微受傷，須先傷口處理→戴乳膠手套→洗手擦乾才可繼續；未照此流程者，**自受傷後所操作之部分一律不予計分**；因處理傷口延遲之時間不另補。
2. **生命徵象測量 評審表 備註三（複測填寫方式）**：初/複測數值需分填「應檢人初測／複測」「監評人員初測／複測」欄位，並記錄於《生命徵象測量》紀錄單簽名。
