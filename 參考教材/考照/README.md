## 學科三大提醒
1. **學科題庫是公開免費的**（勞動部技能檢定中心公告），不必買書即可刷題。
2. 學科除專業科目外還考**共同科目**（90006 職業安全衛生、90007 工作倫理與職業道德、90008 環境保護、90009 節能減碳）——尚未下載，需要時可另抓（同官方系統）。
3. 每次準備前回官方清單確認有無更新版（比對版次/公告日期）。

## 術科關鍵提醒
- 術科照**評審表逐項打勾扣分**，步驟會改版 → **一律以官方現行應檢資料為準，不信網路影片**（影片不會跟著改版）。
