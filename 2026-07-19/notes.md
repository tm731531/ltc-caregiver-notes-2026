## 🎯 考試重點速記（背這區）

### A. 照服員角色、工作界線與法律責任

**實務四原則：保密 → 隱私 → 報告 → 紀錄**（講師明確依此順序講）

- **照顧紀錄要保存 7 年**。出事時法院**第一時間查扣照顧紀錄**；紀錄空白＝難以脫責，詳實＝有保障。
- **告知病情不是照服員的職責，是醫師的職責**。洩漏案主病情可能被告**妨害名譽罪**。
- **被照顧者說「不要」就要停止服務**——他有說不的權利。5～10 分鐘後再問，仍拒絕就告知督導。
- 擦身體、換尿布、洗澡一定要**拉布簾或關門**，即使對方失智也一樣。
- **居服員不能做侵入性行為**：抽痰、灌食、挖便、傷口處理。（家屬可受訓自行執行）
- **被誣賴偷竊時，任何人不得搜身、搜包包**，須報警或督導到場。
- **不要去督導其他照顧服務員**——那是護理人員的職責（講師說「這是最重要的」）。
- **不接受醫師的口頭或電話醫囑**，要把電話轉給護理人員。
- **沒有醫師處方或護理人員指導，絕對不要給藥**；不協助插管、拔管。
- **不要單獨執行需無菌技術的護理行為**；但護理人員在場要求協助時應幫忙。
- 發現他人**虐待、遺棄、傷害**老人，一定要報告護理督導；不處理就往上報到院長。
- **糖尿病或灰指甲**修剪指甲前，須經**護理人員評估**（與 07-18 一致）。
- **身體約束**必須經**醫師醫囑 ＋ 家屬同意**，且需定時監測紀錄。未經醫囑與同意的約束＝**妨害自由**。

**法院判例（易考的過失型態）**

| 判決 | 情境 | 關鍵原則 |
|---|---|---|
| 有罪 | 家屬探視時照服員去做別的事，小孩推倒輪椅致死 | **視線不能離開被照顧者**，即使家屬在場 |
| 有罪 | 山坡地推輪椅未檢查煞車、推法錯誤 | 下坡**老人家要在前面**；推之前**先檢查煞車**。判業務過失致死 |
| 有罪 | 買麻糬給老人，整塊吞下噎死 | 給老人吃東西要**切細、切小塊** |
| 有罪 | 帶長者上完洗手間就轉身清廁所，長者跌倒致死 | **先安頓好**再回頭清理 |
| 無罪 | 糖尿病患未每日測血糖 | 照顧手冊規定「**有打胰島素才須每日測**」，該案主未打 → 無因果關係 |
| 無罪 | 躁動症者洗澡撞水龍頭致死 | 「**背朝水龍頭**」的規定是**事後才補上**的 → 必須事發當時已有規定才能課責 |

> 附帶責任：刑事之外還有**民事連帶賠償**（殯葬費；死者子女未成年者另有扶養費）。

**工作範圍與拒絕**
- 不屬於居服範圍：掃天花板蜘蛛網、拖床底、幫全家煮飯、全家洗衣、掃門口狗大便、天天洗全家共用的抽油煙機。
- 婉拒流程：**委婉說明範圍 → 不與案家爭執 → 仍堅持則通報督導出面協調**。案家真有需求可**再申請**。
- **性騷擾高風險地點**：被**案主**騷擾多在**浴室**（協助沐浴時）；被**案家屬**騷擾多在**客廳**。應明確拒絕＋寫入紀錄＋告知督導。

**照服員的角色（六種）**：服務者、保護者、代言者、陪伴者、關懷／安慰者、資源轉介者。

**服務場域**
- **機構式**：**護理之家**收「三管」（鼻胃管、導尿管、氣切）、**無年齡限制**；**安養／長照機構**收「兩管」，受**老人福利法**規範、原則 **65 歲以上**。
- **社區式**：日間照顧、日間托老、社區關懷據點（最多）、居家護理、**家庭托顧**（在自己家中照顧，人數有上限）。目的是**在地老化**。
- **居家式**、**醫療院所**、**退輔會系統**（榮民醫院、榮家，已開放一般民眾）。

**長照團隊職責**：護理師（**對照服員負監督指導責任**，工作時只接受護理督導指揮）／社工（評估、活動、資源）／醫師／營養師（老人餐食**少鹽少油少糖**）／物理治療師（關節活動、肌力）／職能治療師／藥師／語言＋呼吸治療師。**會計師不是長照團隊成員**（考題）。

**其他**：長照專線 **1966**；長照需求定義為**需照顧半年以上且無法恢復**；**失智在法律上以法院監護宣告為準**，宣告前其行為與一般人同樣有效。

### B. 人際關係與溝通技巧（單元 9）

- **溝通的定義**：要讓**對方願意把心裡的話講出來**才叫溝通；只是對方聽命令，那叫命令不叫溝通。（講師全段重複強調）
- **溝通的 12 個絆腳石＝「一打髒話」**（無形殺手，投影片＋講師各講兩次，最完整的考題結構）：
  **①批評 ②嘲笑 ③諷刺 ④評價性讚美 ⑤命令 ⑥威脅 ⑦說教 ⑧不當的詢問 ⑨忠告 ⑩安慰轉向 ⑪建議 ⑫保證**
  - 評價性讚美例：「你一向最勤勞，請幫我——」（讚美是為了要對方幫忙）
  - 安慰轉向例：「不要傷心了，聽說您最近升官了！」
  - 保證例：「保證下個男人會更好！」
- **消極聆聽 vs 積極聆聽**
  - **消極聆聽**：爭吵時不說話。比當面衝突好，但**問題沒有解決**。
  - **積極聆聽**：讓對方說出「**對啊！**」「**沒錯！**」、心裡覺得「就是如此」。
  - **有效溝通第一步**：①對方有火氣時先**降火** ②對方沒情緒時，讓他有**被瞭解的感覺**。
- 「說、聽、讀、寫」四種溝通行為中，**「聽」佔了最大部分時間**。
- **四不一要原則**（有效溝通）：不要讓對方覺得受**威脅**／不要把氣氛**搞僵**／不要**訴諸恐懼**／不要試著**一次**就完全改變對方立場／**要有同理心**。
- **四不**（態度）：不批評、不責備、不抱怨、不說教。
- **情緒中不要溝通，尤其不能做決定**——情緒中做的決定都是錯的，要暫時離開現場。
- **四類長者特質的應對**（典型考題結構）：

| 類型 | 特徵 | 應對方式 |
|---|---|---|
| **管家婆型** | 好管閒事、多嘴多舌 | ①勿把批評當侵害而自衛 ②問清楚避免斷章取義 ③運用傾聽 ④**請老人家提供解決方法** |
| **指揮官型** | 倚老賣老、擅權跋扈 | ①真誠與尊重 ②**讓他有表現機會**（戴高帽）③請他的**老朋友**幫忙溝通 ④書信電話等間接方式 |
| **活在回憶型** | 反覆回憶過去輝煌 | ①傾聽 ②正向引導與分享 ③用他熟悉的方式傳遞訊息 |
| **渴求關懷型** | 高度親密需求 | ①安排家庭聚會 ②電話、書信、卡片問安 |

- **與高齡者互動三原則**：**同理心**（思想感情上與長者合為一體）、**尊重**（受尊重本身即可引發內在成長潛能）、**關懷與接納**（就算是難纏的長者也要真誠）。
  - 高齡者從**退休開始**會陸續感受到：失落、寂寞孤獨、無能無助、**死亡焦慮**、希望被尊重；**晚年入住機構更明顯**。
- **關係建立與有效溝通之原則**：用**相同的語言**／說話**清晰音量足夠**／**尊重與平等**／注意**口語與非口語**反應、注視對方／不冠**刻板印象**／別急著發言，作**忠實的聽眾**／傾聽與同理心／不為迎合對方而**指摘第三者**／了解對方個性習慣與**過去生活背景**／注意對改變的**承受力**／引導其說出心中想法／**回覆示教**（做做看）。
- **非語言溝通**三組成：**身體語言**（面部表情、目光接觸、身體動作姿勢手勢、**觸摸行為**）、**副語言**（音調、音量、節奏、抑揚頓挫、沉默）、**個人空間／人際距離**。
  - **臉部表達的情緒全世界都一樣**：驚訝、快樂、生氣、厭惡、輕視、害怕、悲傷。
- **專心傾聽的三大障礙**：**急於發言**（怕忘了／被講完、以自我為中心、愛表現）、**過度講究邏輯**（忽略情感）、**帶有偏見**（預設立場、因人而言）。
- **加強聽的風格**：避免以批評態度對待說話者／聽出**言外之意**／注意**提問時間**是否適當／**避免代替對方解決問題**或替他思考／注意非語言溝通。
- **要聽五種聲音**：朋友、同事、主管、教師、**不同意見人**。
- **情緒管理**：生氣時**少說不說**／不了解時不說／與此事無關者不說（**不翻舊帳**）／**暫不說話也是一種溝通**／勿遷怒。
- **衝突處理**：溝通過度劇烈、不舒服焦慮時最好**中止或休息**，**等有妥協機會時再商量協調**。**謹慎批評，慷慨讚賞**。
- **說話原則口訣**（投影片「該怎麼說 1／2」，易考）：

| | |
|---|---|
| 急事 → **慢慢**的說 | 討厭的事 → **對事不對人**的說 |
| 大事 → **清楚**的說 | 開心的事 → **看場合**說 |
| 小事 → **幽默**的說 | 傷心的事 → **不要見人就說** |
| 沒把握的事 → **謹慎**的說 | 別人的事 → **小心**的說 |
| 沒發生的事 → **不要胡說** | 自己的事 → **聽聽自己的心**怎麼說 |
| 做不到的事 → **別亂說** | 現在的事 → **做了再說** |
| 傷害人的事 → **不能說** | 未來的事 → **未來再說** |

- **謙恭四層次**：對**長輩**謙恭是**本分**／對**平輩**謙虛是**和善**／對**晚輩**謙遜是**高貴**／對**所有人**謙恭是**安全**。
- **長照現場守則**：**千萬不要與個案或家屬發生爭執**（講師說「非常重要」，重複兩次）／機構有事**第一時間通知家屬**／**不要碰觸個案財產**（拔戒指飾品有竊盜之嫌）／照顧身障者**不要表達同情**，他要的是**同等的待遇**。

### C. 認識失智症與溝通技巧（單元 5）

**定義與分類（以投影片／課本為準）**
- 失智症＝**多重智能缺損**，含記憶力與其他智能障礙（語言、感覺與動作整合、執行功能、整體注意力、社交認知），**且程度足以影響日常生活功能**。
  - 講師強調：定義中**最重要的一句是「日常生活不能執行」**。
- **DSM-5 更名為「認知障礙症」（major neurocognitive disorder）**。
- **分類與佔比（投影片數字，考試以此為準）**：
  - **退化性失智症 50～60%**（阿茲海默症最大宗，好發 65 歲以後）
  - **血管性失智症 10～20%**（第二大原因；腦中風或慢性腦血管病變致腦細胞死亡，亦稱**多發性腦梗塞失智症**；現傾向稱**血管性認知功能障礙**）
  - **混合性 10%**
  - **其他類型 10%**
- **額顳葉失智症**：進行性、退化性腦部疾病，常發生於 **60 歲以下**，是**早發性失智症最常見的原因**。
- **路易氏體失智症**：注意力時好時壞、**快速動眼期睡眠行為障礙**、**視幻覺**、似巴金森氏症動作障礙——**符合兩項即可確診**。

**診斷與評估**
- 診斷原則：**緩慢進行、目前仍無法治癒**；早期發現可延緩病程、維持功能與生活品質、減輕照顧者壓力。
- **早期診斷的主要目的是讓病人與家屬趁早做好準備**，降低誤解與衝突、預防危險。
- 三個重要部份：**臨床診斷、病因探討、找尋可治療問題與危險因子**。
- 評估須包括：生活功能、精神狀態、社會功能、營養、慢性病、藥物、家人與社會支持；若有**周全性老年醫學評估（CGA）**更佳。
- **篩檢工具**：**MMSE**（簡易智能檢查）、**AD8**（極早期失智症篩檢）、**SPMSQ**（簡易心智狀態問卷）、**畫鐘測驗**、**Mini-Cog**、**GPCOG**、**MIS**。
  - 講師補充：居服單位最常用 **AD8 與 SPMSQ**；**MMSE 較複雜、較少用**。
- 評估重點：**定向感（人、時、地、物）、語言、結構（建構）能力、ADL 與 IADL**。

**三個 D 的鑑別（表 4-2，最易混淆考點）**

| 項目 | 失智症 dementia | 譫妄 delirium | 憂鬱症 depression |
|---|---|---|---|
| 症狀發生 | 潛伏且無法察覺 | **突然發病** | 某種程度可察覺 |
| 持續時間 | 數月到數年 | 數天到數週 | 數週到數月 |
| 病程 | 惡化速度緩慢 | **暫時性** | 急速惡化 |
| 警醒度 | 正常 | 多樣化 | 正常 |
| 注意力集中 | 最初是正常 | 多樣化 | 無法集中 |
| 思考 | 窮困 | 缺乏組織性 | 緩慢 |
| 記憶力問題 | 近期記憶受損 | 近期記憶受損 | 記憶力下降、判斷力變差 |

**失智 vs 正常老化（表 4-1）**：失智者是**所有的經驗**都忘、**漸進性**、**經常**無法遵循指示、**漸漸不行**使用標誌備忘、**漸漸喪失**算數與自我照顧能力；正常健忘只是**部分**、**偶爾**、**通常可以**。

**症狀三大類**：**認知退化、行為異常、精神情緒問題**（含個性改變）。
- **攻擊性**：情緒一來就可能攻擊人，照顧時要**隨時提高警覺**，不要以為此刻心情好就沒事。
- **不喜歡洗澡**。
- **食慾改變**：可能一直吃、吃不飽（講師解釋為飽足中樞受損）→ 給容易有飽足感的食物。
- **典型特徵：以前的事記得住，現在的事記不住。**

**日常生活照顧目標（投影片，六項）**
①預防疾病與**合併症** ②安全維護與**預防意外** ③滿足**休息與活動**需求 ④症狀**緩解與控制** ⑤維護增進**自我照顧能力** ⑥維持**正常身體功能**。（最主要目標：**維持日常生活功能**）

**照顧要點（講師實務）**
- **排尿照顧**：先分辨是「尿不出來」還是「不知道廁所在哪」。**先尿尿再喝水**（尿完心理壓力小才願意喝）。不要直接問「要不要尿尿」，改說「起來走一走」直接帶到馬桶坐。**不要輕易包尿布**（退化更快）；非包不可就**內褲→紙尿褲→外褲**。床單用**防水**材質。**發現尿濕不責備、不教訓**。
- **飲食**：冰箱**定期清理**／**定時**進食／**不要催促**／餐具與食物**顏色分明**／熟悉口味、**簡單餐具**／握力不好把**握柄加大**／不想吃不硬餵／食物**不要太燙**（可能感覺不到而燙傷）／**食慾改變第一件事是檢查牙齒**。
- **安全**：醬油、藥品都要收好（案例：把整罐醬油當可樂喝完）。失智者所在樓層**頂樓門要鎖**。
- **維持功能**：讓他做一點家事（如剝蒜頭）並給回饋；**不要當他的面把他做的事推翻重做**。
- **遊走**：掌握作息規律 ＋ **門上掛鈴鐺**。
- **「失智不是呆子，不要把他當呆子看。」**（講師重複三次）

**與失智者溝通（考點密集）**
- **給封閉式二選一，不要開放式問句**：問「要吃飯還是吃麵」，不要問「今天中午想吃什麼」。
- **不要問「你知道我是誰嗎」**，直接告訴他你的名字——問了只會**增加他的失敗感**。
- **指令要具體明確**：說「水杯拿過來」，不要說「那個東西拿過來」。
- **一次只講一件事**。
- **不要對嗆、不要爭辯、不要拿證據**，要**轉移話題／轉移焦點**（錢掉了的示範：不否認、不發誓，改帶他去看冰箱，看完就忘了）。
- 重複問同一句話要**當作第一次聽到**，不要打斷、不要說「你已經講過了」。
- 出現幻覺說看到已故親人時，**不要糾正他**。

**大腦結構（講師以講義第 9 張大腦圖教學）**
- **額葉—情緒**（講師強調最重要）／**顳葉—聽力**／**頂葉—判斷**／**枕葉—視覺**；皺褶處即**大腦皮質**。
- **腦細胞壞了不會恢復**。
- 應用：個案脾氣暴躁是腦細胞在驅使他，不需要罵回去——「知其然，知其所以然」。

**失智為什麼特別被重視（講師說課本沒寫）**
**失智 ⇒ 很快就失能**。糖尿病、高血壓的病人還能自己去看醫生；失智者必須有人在旁，那個人原本在工作賺錢 → **一個失智的人，等於多了一個不能生產的人**。

### D. 臨終關懷及認識安寧照顧（單元 19）

**⭐ 全單元最核心的一句：人死亡時「聽覺」最後消失。**（講師在 719.7 重複四次以上）

**核心精神**：**以尊重生命的態度，達到「生死兩無憾」**——活著的人沒有遺憾，往生的人也沒有遺憾。（講師說「整個安寧照顧只講這一句話」）

**起源與發展（投影片，考點密集）**
- **Hospice** 最早在**羅馬時代**，是照顧旅客及病人臨終的地方；中世紀歐洲**修道院**附設安寧院，作為朝聖者、旅行者**中途休息補足體力的驛站**。
- **桑德絲（Dame Cicely Saunders）**：1918 生於英國 → 1940 護士 → 1947 社工 → 1947 照顧年輕癌症病人**大衛**結緣 → 1958 成為醫師 → **1967 年於英國設立世界第一家安寧院「聖克里斯多福」**。
- **台灣**：緣起於**趙可式博士**推動**安寧居家療護**。**1990 馬偕 → 1994 耕莘 → 1995 台大**。**1996 年**公告「緩和醫療病房設置規範」及「安寧居家療護設置規範」。
- 台灣三種用詞同源：**安寧療護、緩和醫療、安寧緩和**。

**服務型態（投影片）**
安寧療護**住院**服務／**居家**服務／**共同照護**／**預立選擇安寧緩和醫療意願**（免除 CPR 的折磨）。**2013 年 1 月 9 日再次修法**通過，讓末期案主家屬可選擇**終止或撤除維生系統**。

**⭐ 安寧照顧 ≠ 安寧病房**：**隨時隨地都可以做安寧照顧**，一般病房、居家服務都會做到，不是只有安寧病房才有安寧照顧。（講師重複強調兩次；719.7 又說「內科病房可不可以做安寧？可以，記得哪一個地方都可以做安寧」）

**進安寧病房的意義**：**不做積極治療，但不放棄症狀治療**——傷口仍換藥、**最主要是治療疼痛**（嗎啡、疼痛貼片），仍給低濃度氧氣。

**臨終關懷的特殊議題（投影片）**
- **先前預立遺囑及預立代理人**：有決定能力的成人，在尚未喪失決定能力前，預先對其醫療做出指示的文件。
- **不予急救（DNR）**：疾病末期無法復原，且死亡是可預期者。
- **維生治療的不予及撤回**：拒絕侵入性維生治療。

**⭐ 臨終病人心理調適五大階段（Kübler-Ross，必考）**
**震驚與否認 → 憤怒 → 討價還價 → 憂鬱 → 接納**
- 實際心理狀態錯綜複雜，矛盾情緒會**交替出現**：否認↔承認、恐懼↔勇氣、討價還價↔準備、悲憤↔平靜、憂鬱↔淡漠、絕望↔希望、不甘心↔認命。

**照服員在安寧照顧的功能與角色（投影片）**
技能照顧功能／情緒支持功能／行政支援；**支持者**（傾聽、接納、善解人意、溝通、分享分擔感受、尊重信仰）／**供給者**（付出愛心、幽默感）／**自我肯定者**／**遵守倫理者**（保守病人隱私）。

**協助臨終病人及家屬面對死亡（投影片）**

| 對**臨終病人** | 對**家屬** |
|---|---|
| 情緒疏導與支持 | **分擔照顧的責任** |
| 維持良好的溝通關係 | 情緒疏導與支持 |
| 經濟上實質的幫助 | 維持良好的溝通關係與社會功能 |
| 統整生命意義 | 經濟上實質的協助 |
| **達成心願** | **哀傷輔導** |

**⭐ 臨終生理徵象與處置（講師實務，考點密集）**

| 徵象 | 處置 |
|---|---|
| **四肢冰冷** | **千萬不要用烤燈！** 血液已集中回重要器官，烤燈會把血拉回四肢，死得更快。**只要加棉被，一條不夠加兩條** |
| 脫肛、大量解便 | **一定要清乾淨**，維持外觀有尊嚴，讓家屬慢慢接受 |
| 喉嚨發出聲音（痰音／死亡嘎聲） | 表示很快要走了。**但聽覺仍然很清晰** |
| 看到已故親人、幻聽 | **不要糾正他**。回應「我知道你媽媽來看你」，不要說「你媽早就死了」 |
| 不吃東西 | 吃不下就不要勉強餵，免得吐出來 |
| 無尿 | 尿量是重要判斷指標 |
| 呼吸起伏大、喘一陣停一陣 | — |
| 睡眠時間越來越長、意識改變 | — |

**⭐ 臨終溝通的禁忌與話術**
- ❌ **「早一點送來就好」——這句話永遠不要說**。家屬會永遠痛，且對事情一事無補。
- ❌ 家屬有遺憾時，不要說「你不知道我跟你爸有多好」（＝一百支箭插在他身上）；也不要建議「去他的靈堂看他」（等於承認沒見到最後一面）。
- ✅ **對家屬的標準話術**：「**你該做的都做完了，你已經做得很盡力了，你沒有別的事可以做了。**」
- ✅ **不會講好話就只要陪伴**，並經常說「**我就在你身邊**」。
- ⭐ **討論後事一定要隔到另外一個房間**——不可在床邊講、不可只隔布簾，因為**他會聽到**。
- 對兒童告知病情：**先問「你家裡還有什麼人？」**（講師說這句超重要），不可直接問「你爸爸呢？」以免戳到痛處；再用他能懂的概念解釋，並**給他一個做得到的任務**。

**⭐ 臨終照顧的原則（結尾快問快答）**
- 需要餵食嗎？→ **不用，不想吃就不要勉強**。
- 要插鼻胃管嗎？→ **不用；如果他拒絕，我們就不要做**。
- 癌症病人想吃某樣東西？→ **想吃就給他吃**。
- 個案不想再吃藥打針但家屬堅持？→ **要去說服家屬**，**依照病人的希望去做**。
- 想回家但家屬不讓？→ 變通：把家裡的枕頭被子搬到病房佈置／去他家**錄影**播給他看。**「我們只要那個感覺，不一定要真正到那個地方。」**

**照顧者自身的壓力調適**
- 臨終案主照顧者的壓力來源：怕死、不知如何告知病情、害怕面對 DNR、害怕自己幫不上忙。
- 講師給的方法只有一個：**讓自己暫時離開**，讓別人接手。
- 核心價值：「**每個人要走的時段都跟你沒有關係，但我們可以在他那個當下幫他做好。**」

### A-1　職涯路徑與資格（719.1、719.2）
- 90 小時結訓後，講師建議**優先到長照機構或醫院**累積經驗，疑難雜症都學得到；有醫院服務證明再轉居家會更被接受。居家服務多為「做中學」，機構則有人在旁督導。
- 具 2 年以上照顧經驗可應徵：衛生局個管、醫院個管師、照管專員、督導。
- **證照考 vs 公務人員考**：證照（如社工師）是**資格考**，平均 60 分即及格，跟自己競爭；公務人員考試**有名額限制**，擇優錄取（若最低錄取 63 分，考 62 分即落榜）。**公職社工師須先考上一般社工師**才能報名。
- 派案流程：家屬打 **1966** → 長照中心派**照管專員**到府評估 → 核定時數與服務項目 → 派居服員到家服務。

### A-2　照顧服務工作內容分類（719.2）
- **基礎生活照顧**：鋪床、生活支援。
- **技術性照顧**（須在護理人員指導下）：尿套、尿管照顧、鼻胃管灌食、熱敷冰敷、異物梗塞處理、**協助**抽痰與氧氣使用（只是協助，不能一個人做）。
- **安全保護照顧**：協助上下床、預防跌倒、輪椅使用。
- **預防性照顧**：測量體溫、脈搏、血壓、呼吸等生命徵象（在護理人員指導下）。
- **醫院內服務**：環境介紹、病床整理、會客協助、**協助遺體清潔**、檢查治療前準備、發現異常立即通報護理人員。
- **原則**：**服務範圍不可涉及醫療與護理行為**，但在護理人員指導下可協助技術性照顧。

### A-3　職業倫理與儀容（719.2）
- **特質**：關懷、同理心、誠實、盡責可信任、有禮貌、**合作**（別人要幫忙時要接受，不要逞強）、熱心、面帶微笑、**自知之明**。
- **儀容**：頭髮乾淨、口腔清潔（下午要照顧就別中午吃臭豆腐大蒜）、**手腳指甲剪短**、每天洗澡換衣避免體味、服裝整齊。
- **尊重生命**：尊重服務對象的宗教、文化、背景與信念，**不能因對方失智或同性戀就拒絕照顧，要一視同仁**。

### B-1　溝通的比喻與案例（719.3、投影片）
- **心路順暢，能量逐湧**：人的血液阻塞會產生心臟病、中風；**溝通正如血管**，阻塞了會產生誤會、衝突、人際關係不好、團體與家庭不和諧。
- **野雁團隊精神**（投影片第 3 張）：蜜蜂、螞蟻及野雁都靠團隊合作把群體力量發揮到最大。野雁排成 **V 字形**，領頭的是體型最大的雁王，彼此拍動翅膀產生的氣流浮力可減少飛行力氣。**成群雁子的飛行距離比單隻提升 71%。**（數字見待查核）
- **移山大法**：「要移山其實很簡單，**如果山不過來，我們就過去**。」——人類可以透過**改變自己的態度去改變自己的生活**。
- **卡內基**：只要你先**主動接近別人**，你在**兩個月內結交的朋友**，絕對比**兩年內**主動接近你的人還多。→ 結交朋友的不二法門是**自己先成為別人的朋友**。
- **詹森總統的兩個瓶子**：一個是「**借出的人情**」，一個是「**積欠的人情**」，他隨時讓「借出的人情」那個瓶子**永遠保持二倍滿**。
- **維持良好人際關係**：在**獲得之前先付出**／想想最近為他們做了什麼／有人**得意**時打電話給他們／在他們**失意**時**更要記得打電話**。
- **職場人際「九不」**：不以為職位就是權力／不錯把組織的人際網當成自己的／不把眼睛所見與確實性混為一談／不做小氣鬼／不對別人說「不」／不只加入一個團體／不過河拆橋／不拋棄離開的人／不貶低個人關係的價值。
- **「十通」（前四項）**：**主動與人交談**（Speak to people）／表現**真摯與熱誠**／時常**微笑**／**叫得出別人姓名**。
- **幽默地說的示範（日本空服員志工）**：長輩問「我可不可以用手摸你一下？」她回答「**當然可以啊，但是你只能用左手摸**」——因為長輩半身不遂，那隻手抬不起來。好處：①不讓長輩難堪 ②給他鼓勵，長輩此後每晚勤練復健。

### B-3　挑選機構的三個觀察指標（719.3）
①**氣味**（門口就有異味＝環境管理不好）②**工作人員與長者的互動**（會不會打招呼問安）③**評鑑成績**（結果公布於縣市政府或衛福部網站）。
> ⚠️ 講師提醒：**評鑑優等不等於絕對好**。案例：一家評鑑優等的機構把賣不出去的過期麵包煮成稀飯給長輩吃，員工檢舉後查證屬實，遭限期改善並罰款。

### C-1　失智的預防與延緩（719.5）
- 講師指定「造成失智的原因／預防失智」那一頁**要畫起來，是最重要的**。
- **均衡飲食**：把過去不吃的東西現在都試著吃一點，因為長期缺乏那種營養素；已是專業人員就不可以再挑食。
- **吃原型食物**；**地中海飲食**（涼拌、堅果核果、橄欖油）與**得舒飲食（DASH）**。
- 經常**閱讀**；生活要**多元**，不要只做單一活動。
- **不要什麼事都幫長輩／孩子做完**，要讓他用腦。

### C-2　失智 vs 精神疾病（719.5）
- **最大差別：失智不會好（持續退化），精神疾病可以控制、可以變好**（按時服藥、規律照顧）。
- **照顧精神疾病者最重要的一件事：幫助他「生活規律化」**，且要**漸進式**（一週有一天正常起床就大力讚美，再進展到兩天三天），不能一開始就要求天天準時。
- 原則：**不要對精神病人失望，也不要對失智個案期望。**
- 正常 vs 病態的界線（如反覆檢查門鎖）：**差別在於我們可以停止，他不能停止**。

### C-3　用個案聽得懂的語言溝通（719.5）
- **豆腐比喻**：阿嬤問「同樣是腦震盪，為什麼有的會死有的不會」→ 用便當店夾豆腐比喻，同樣夾兩塊，有一次啵掉了、有一次完整，讓她理解「損傷程度不同」。
- **顧腦藥案例**：一位國小畢業的年長婦女腦部開刀後須長期服抗癲癇藥，講師說「這是**顧腦的藥，吃了會變聰明**」，她就願意規律服藥。
- 原則：**依照個案的程度講他聽得懂的話**——不是看低他，而是不能只有一套說法。

### D-1　安寧實務案例（719.7，四個貫穿全段的案例）
1. **跳樓學生**：講師任護理長時的病房規定——「**病人救不活的時候，我們就救活人**」。確定救不活就**開放家屬自由進出加護病房**陪伴，不要讓家屬在門外等到開門那刻才被告知。
2. **肝癌媽媽與國小六年級兒子**：先收集資料判斷孩子對死亡的理解 → 用他能懂的數學概念解釋病情 → **給他一個做得到的任務**（大聲呼喚媽媽），既紓解情緒又讓母親即時得到愛。關鍵理由：**人死亡時聽覺最後消失**。對高二姐姐則用「回家拿一套衣服」的方式——「病好了要穿自己的衣服回家，病沒好也要穿自己的衣服回家」。
3. **肺癌病人未竟心願**：病人最後想跟講師見一面，女兒未轉達，事後大哭懊悔。講師當場處理：「**你跟我講完，我就跟你爸見完面了。所以我們兩個是有見面的。**」——直接解決「沒見到面」這件事本身。
4. **副院長灌冰水自責**：先確認母親生前**沒有任何侵入性治療** → 導出「你媽媽是求仁得仁」 → 重新框架：「**可以放冰水這件事，不是你說的，那是你媽說的**，她借你的嘴巴告訴別人，她就可以不用再受這些苦。」 → 立刻給他一件事做。**不能在電話裡表現冷漠**。
5. **實習生自責案例**：學生幫病人抽痰後病人過世，認為是自己害死的。講師先解釋抽痰不是致死原因，仍無法釋懷就**帶他去幫阿嬤做大體照顧**（墊枕頭、蓋棉被，對阿嬤說「衣服都換好了，也給你枕頭、蓋好棉被，你可以安心」）。

### D-2　癌末與轉移（719.6，講師定義，見待查核）
- **癌末＝第四期＝末期**。講師定義：**兩個以上器官出現同一種癌細胞**。
- **轉移**的判準：在不同器官驗到**同樣的癌細胞**（肺→腦、乳房→胃）。**原發部位大概一粒，續發部位大概兩粒以上。**
- 同時罹患好幾種癌但彼此非轉移者，各自都是原發，**不算轉移**。
- **末期還不會到瀕死**；瀕死＝馬上就要死亡。
